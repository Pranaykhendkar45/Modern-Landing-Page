# 🚀 Animated Landing Page

A modern and interactive landing page built using **HTML, CSS, JavaScript, GSAP, and Shery.js**.

This project was created to explore advanced frontend concepts such as smooth animations, image transitions, interactive effects, and modern UI design. The goal was to create an engaging user experience while learning industry-standard animation libraries and frontend development techniques.

---

## 🌟 Features

* Smooth GSAP Animations
* Interactive Shery.js Image Effects
* Dynamic Hero Section
* Animated Text Transitions
* Modern Landing Page Design
* Responsive Layout
* Interactive Navigation
* Clean and Minimal UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* GSAP
* Shery.js
